# Templates

Copy a stack pack into your project, then adjust commands to match the repo.

```bash
cp templates/nextjs/AGENTS.md /path/to/project/AGENTS.md
cp templates/nextjs/.grokignore /path/to/project/.grokignore
```

Available packs:

- `python-fullstack`
- `fastapi`
- `nextjs`

Each pack includes:

- `AGENTS.md` with stack-specific agent instructions;
- `.grokignore` with safe defaults;
- `README.md` with recommended skills and first prompts.
